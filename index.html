WizardSkillz

This application allows users to trace spell movements for several famous spells found in the Harry Potter franchise. Users can save drawn spells in order to compare their own progress. There is an affordance to add notes to the tracings and edit those notes. Users can delete old spells as well. The best feature allows users to cast a spell without a guide to see if you have mastered each spell.
